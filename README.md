# poker

## 概要
C言語で作成したWindows向けポーカーゲーム。

## 機能
- exeをダブルクリックで起動（要Windows）
- ポーカーゲームが可能
    - 残すカードにチェックを入れてドローボタンを押下
    - 軽量化のため画像を使用しておらず、各スイートは以下で代用
        - クローバー：■
        - ダイヤ：◆
        - ハート：●
        - スペード：▲
        - ジョーカー：★
    - ジョーカーの枚数は0～2枚の間で変更可能

## 使用技術
- 内部仕様：C言語
- GUI：WIN32API

## コンパイルコマンド

>gcc poker.c -mwindows

## 注意点
- 文字コードは「Shift JIS」じゃないと文字化けするので注意
